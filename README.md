# NOT DEFTERİM 
## AÇIKLAMA 
Bu proje, C# ve Windows Forms kullanılarak geliştirilmiş basit bir Not Defteri (Notepad) uygulamasıdır. Kullanıcıların hızlı bir şekilde metin yazmasını, düzenlemesini ve kaydetmesini sağlayan pratik bir araçtır. Amacı; sade, hafif ve kolay kullanılabilir bir metin düzenleme deneyimi sunmaktır. Windows’un klasik Not Defteri uygulamasına benzer şekilde çalışır ancak geliştirilebilir modüler bir yapıya sahiptir.
### ÖZELLİKLER
* Yeni dosya oluşturma  
* Mevcut dosyayı açma  
* Dosyayı kaydetme / Farklı kaydetme  
* Kes / Kopyala / Yapıştır  
* Yazı tipi değiştirme  
* Basit ve anlaşılır arayüz
### KULLANILAN TEKNOLOJİLER
* C#  
* .NET Framework / .NET Windows Forms  
* Visual Studio
